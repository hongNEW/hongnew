✍️ HTML

📝 약간 메모장 느낌, 어떤 내용이 들어갈지 스케치한다고 생각하면 될 듯.

✔️ !DOCTYPE html
: 문서 타입 선언, "html5로 만든 웹페이지임"이라고 알려주는거, 제일 처음에 입력하면 됨.

✔️ hmtl
: html의 시작과 끝. "!DOCTYPE html" 다음에 들어오고, 문서 제일 마지막에 배치 됨.

✔️ head
: 브라우저의 정보를 나타냄.
주로 title(탭 이름), meta(언어...??? ex. meta charset="utf-8"), link(css연결), script(JS연결)가 이 사이에 들어옴.

✔️ body
: 페이지에 실제로 보이는 내용을 넣는 곳. (텍스트, 이미지, 링크, 메뉴 등등 다 여기 들어감.)
body는 한 페이지에 한 개만 들어감.

✍️ CSS

📝 html로 스케치를 한 후에 색깔을 입혀주는 거라고 생각하면 될 것 같음.
지금보니 여태 색칠하고, 글자 굵게 만들고 했던 것들이 다 CSS였던 것 같음.
챗GPT하는 말이 CSS는 한 곳에 모아두고 관리하는게 편하다고 함

✔️ 선택자
: 특정 태그 전체에 적용되는 속성
예를 들어 P는 모든 P 태그에 적용.
".****"태그는 class="****"태그에만 적용. ****은 내가 이름짓기 나름. 약간 문단 개념으로 생각하면 될 것 같음.
